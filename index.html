Using on iPad / Multiple Devices

This app saves data in this browser's local storage only, which does not sync across devices.

  1. Open this HTML file (e.g. from Google Drive via the Files app).
  2. Use Open from Drive at the top to load the latest saved data file.
  3. Make your changes.
  4. Use Save to Drive and, when prompted, save the downloaded file back into your Drive folder (overwriting the old one).

Don't rely on this browser's local storage alone to carry data between devices — always Save to Drive when you're done, and Open from Drive when you start on a different device.

Quick tips
  • Use Save to Drive (or Export JSON) to download a backup file. On iPad, save it into your Drive folder via Files. If nothing happens, check your browser’s download/popup blocker.
  • Use Open from Drive (or Import JSON) to load a saved file — you'll see a preview before anything is replaced.
  • Search box filters the currently active tab.
  • Data lives in this browser's storage only; it does not sync across devices on its own — always Save/Open to keep devices in sync.
  • Offline use: This app works without internet once loaded. Data is stored locally. Only address lookup and initial loading require a connection.
