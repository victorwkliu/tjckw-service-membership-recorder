How to Use This App

This app uses a shared church record stored in Google Drive. Follow these steps to avoid missing or overwriting another person’s work.

1. Connect to Google Drive

Make sure Google Drive for desktop is running and signed in with the authorized church Google account.

Authorized account
tjckw.cb@gmail.com

You will need the proper password and verification method. For security, the password must never be displayed or stored inside this app.

2. Open the shared church record

Click Import from Drive (Open Shared Record) and go to:

Google Drive folder
Google Drive → My Drive → Service&Membership Record

On the church computer, Google Drive may appear as the H: drive.

Select the church record file and click Open.

3. Follow newer-record warnings

After the shared record has been opened, the app will automatically check whether a newer version is available.

Newer-record warning

If you see “A newer record is available” (or a similar yellow warning), open the newer record before entering or changing information. This protects updates made by another user.

Users do not need to manually compare file dates unless the automatic check is unavailable.

4. Add or update information

After the latest record is open, you may:

  • Add a service record
  • Add a member
  • Update existing information
  • Use Quick Add

Review the information before saving.

5. Save your work

When finished, click Save to Drive (Save Updated Record).

Save the file back to:

Google Drive folder
Google Drive → My Drive → Service&Membership Record
Save confirmation

Do not close the app until the header status shows a green Saved message (changes saved successfully). If it shows Unsaved changes, click Save to Drive again.

Important notice

Never ignore a newer-record warning. Opening an older copy may cause another person’s changes to be lost.

If you cannot sign in, cannot find the shared folder, or are unsure which file to open, stop and contact the church administrator before entering information.

Troubleshooting

Google Drive is not shown

Make sure Google Drive for desktop is running and signed in with the authorized church account. Wait a moment, and then try again.

The shared folder is missing

Confirm that you are signed in with the correct authorized account. If the folder is still missing, contact the church administrator.

The app asks me to open the file again

The file may have been moved or renamed, or this computer may no longer have access to it. Click Import from Drive and select the file again.

A newer record is available

Click Import from Drive (Open Newer Record) before making any changes. Do not continue with the older copy on this computer.

I am not sure whether my work was saved

Look at the app’s record status in the header. Do not close the app unless it says Saved (changes saved successfully). If it says Unsaved changes, click Save to Drive.
